Windows

MacOS/Linux:

  • Install dependencies: pip install -r requirements.txt

  • Run python keras_infer.py --img-mode 0 --video-path 0 if you are using webcam.

  • Run python keras_infer.py --img-mode 0 --video-path $video_path if you are using video.

  • Run python keras_infer.py --img-mode 0 --img-path $img_path if you are using image.
